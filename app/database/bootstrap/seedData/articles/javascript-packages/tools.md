# NPM packages

```
npm i (x) -s ...

eruda
Good console logging in any environment

date-fns
Pure functions that make working with JS date objects easier

lodash
Helper functions

chalk
Used to style terminal output
chalk.blue('message')
chalk.blue.bgRed.bold('message')objects

agenda
Cron-like task runner for NodeJS process
agenda.define('delete old users', (job, done) => {});
await agenda.every('*/3 * * * *', 'delete old users');

forver
Similar to supervisor

fs-extra
Like fs, but with more features

gm
NodeJS wrapper for imagemagick

madge
Generates visualisations of dependencies. Can be used to find circulars.
madge --circular ../dir

jest
Popular, modern, well-documented javascript unit testing framework

request
nodeJS HTTP library

request-promise-native
Promise based wrapper around request, the nodeJS HTTP library
```
